## 📖 À propos

Ce dépôt contient l'ensemble de mes fichiers de configuration (dotfiles) pour créer un environnement de développement cohérent et reproductible sur plusieurs machines. Que vous configuriez un nouveau Mac ou une distribution Linux, ces dotfiles vous permettent d'être opérationnel en quelques commandes.

## ✨ Fonctionnalités

- 🍎 **Compatible macOS** - Configuration optimisée pour les systèmes Apple
- 🐧 **Compatible Linux** - Fonctionne sur les principales distributions
- ⚡ **Installation simplifiée** - Utilisation de `make` pour automatiser le déploiement
- 🔄 **Synchronisation facile** - Gardez vos configurations à jour sur toutes vos machines
- 📦 **Modularité** - Organisation claire des fichiers de configuration

## 🚀 Installation rapide

### Prérequis

- Git installé sur votre système
- Make (généralement préinstallé sur macOS et Linux)

### Première installation

```bash
# Cloner le dépôt
git clone https://github.com/xaviergregor/dotfiles.git

# Se déplacer dans le répertoire
cd ~/dotfiles

# Lancer l'installation
make
```

## 📋 Utilisation

Une fois installé, votre environnement est configuré avec toutes vos préférences personnalisées. Les dotfiles sont liés symboliquement depuis le dépôt vers votre répertoire home.

> **Note**: Consultez le `Makefile` pour voir toutes les commandes disponibles.

## 🔧 Personnalisation

Pour adapter ces dotfiles à vos besoins :

1. **Forkez** ce dépôt
2. **Modifiez** les fichiers de configuration selon vos préférences
3. **Commitez** vos changements
4. **Profitez** de votre environnement personnalisé !

> **Note**: La structure exacte peut varier. Consultez le dépôt pour voir tous les fichiers disponibles.
